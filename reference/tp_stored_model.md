# Model file names stored by user in [base::tempdir](https://rdrr.io/r/base/tempfile.html) using [tp_write](http://treeppl.org/treepplr/reference/tp_write.md)

Provides a list of all the model file names currently stored in
[base::tempdir](https://rdrr.io/r/base/tempfile.html). To verify if
there is a matching data file, see
[tp_stored_data](http://treeppl.org/treepplr/reference/tp_stored_data.md).

## Usage

``` r
tp_stored_model()
```

## Value

A list of model file names.
