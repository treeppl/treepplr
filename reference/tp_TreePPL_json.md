# Convert a tppl_tree to TreePPL json str

`tp_TreePPL_json` takes an tppl_tree create with
[tp_phylo_2_tppl_tree](http://treeppl.org/treepplr/reference/tp_phylo_2_tppl_tree.md)
and returns a list ready to be exported with JSON

## Usage

``` r
tp_TreePPL_json(tree)
```

## Arguments

- tree:

  an object of class json.

## Value

A TreePPL json str
