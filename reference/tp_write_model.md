# Write out a custom model to tp_tempdir()

Write out a custom model to tp_tempdir()

## Usage

``` r
tp_write_model(model, model_file_name = "tmp_model_file")
```

## Arguments

- model:

  A string containing the entire TreePPL code.

- model_file_name:

  An optional name to the file created

## Value

The path to the file created
