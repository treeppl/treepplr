# Create a flat list

`tp_list` takes a variable number of arguments and returns a list.

## Usage

``` r
tp_list(...)
```

## Arguments

- ...:

  Variadic arguments (see details).

## Value

A list.

## Details

This function takes a variable number of arguments, so that users can
pass as arguments either independent lists, or a single structured list
of list (name_arg = value_arg).
