# Convert phylo to a tppl_tree

`tp_phylo_to_tppl_tree` takes an object of class `phylo` and returns a
tppl_tree.

## Usage

``` r
tp_phylo_to_tppl_tree(phylo_tree)
```

## Arguments

- phylo_tree:

  an object of class
  [ape::phylo](https://rdrr.io/pkg/ape/man/read.tree.html).

## Value

A pair (root index, tppl_tree)
