# Data file names stored by user in [base::tempdir](https://rdrr.io/r/base/tempfile.html) using [tp_write](http://treeppl.org/treepplr/reference/tp_write.md)

Provides a list of all the data file names currently stored in
[base::tempdir](https://rdrr.io/r/base/tempfile.html). To verify if
there is a matching model file, see
[tp_stored_model](http://treeppl.org/treepplr/reference/tp_stored_model.md).

## Usage

``` r
tp_stored_data()
```

## Value

A list of data file names.
